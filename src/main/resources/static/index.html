API Documentation

GET /api/sites

List ERW candidate sites with optional filtering and pagination.

Query Parameters:

  • region - Filter by region (optional)
  • sort - Sort field: score, name, id (default: score)
  • dir - Sort direction: asc, desc (default: desc)
  • page - Page number (default: 0)
  • size - Page size, 1-100 (default: 20)
Example: GET /api/sites?region=Madagascar-NW&sort=score&dir=desc

POST /api/sitescore

Calculate site suitability score from environmental factors.

Request Body:

{
  "latitude": -18.15,
  "longitude": 49.40,
  "rainfallMm": 1200,
  "soilPh": 7.2,
  "distanceToRoadKm": 10,
  "basaltAvailabilityIndex": 0.8
}

GET /api/health

Service health check.

About This Platform

This ERW City Viability Analysis Platform is a research tool designed to assess Enhanced Rock Weathering potential across major metropolitan areas and their agricultural hinterlands worldwide. Using 20 advanced parameters based on 2025 climate tech research, it helps researchers, policymakers, and agricultural investors identify optimal locations for ERW implementation.

Target Audience

  • Climate Researchers: Academic and institutional researchers studying carbon removal technologies
  • Policy Makers: Government officials developing climate and agricultural policy
  • Agricultural Investors: Private and public investors in sustainable agriculture and climate tech
  • ERW Companies: Organizations implementing Enhanced Rock Weathering projects

Platform Features

  • 45 pre-analyzed global metropolitan areas with ERW viability scores
  • Interactive world map with detailed metropolitan analysis
  • 20-parameter scoring algorithm based on 2025 industry research
  • Cost analysis and CO₂ emissions calculations
  • RESTful API for data integration
